# Product Service Management System

## About
A learning project built to practice Object-Oriented Programming concepts in C# — including inheritance, encapsulation, interfaces, and design patterns. The application manages products, services, and bundled packages through both a console interface and a WPF desktop UI.

## OOP Concepts Practiced
- Inheritance & Polymorphism — shared base types for products and services
- Encapsulation — domain logic isolated in a reusable Core library
= Interfaces — data access abstracted through a service interface
- MVVM Pattern — clean separation of UI and business logic via ViewModels, Commands, and Validation

## What It Does
- Create, edit, and delete products and services
- Group them into packages
- Inline editing directly in DataGrids
- Import initial data from XML files

## Project Structure
- Core — domain models and business logic (reusable library)
- Proiect — console application
- ProductServiceWpf — WPF desktop UI

## Running the App
- Console: set `Proiect` as Startup Project and run
- WPF UI: set `ProductServiceWpf` as Startup Project and run
